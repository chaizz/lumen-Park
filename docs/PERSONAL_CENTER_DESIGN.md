# Lumen Park - 个人中心 (Personal Center) 产品设计

## 1. 概述
个人中心是用户展示自我、管理内容和查看互动数据的核心页面。它需要清晰地展示用户信息、作品集以及收藏的内容，同时提供编辑资料的入口。

## 2. 页面布局
页面整体采用 **Header + Tabs + Content** 的结构。

### 2.1 头部区域 (Profile Header)
*   **背景图/Banner**（可选）：顶部可配置背景图，增加氛围感。
*   **用户基本信息**：
    *   **头像**：大尺寸圆形展示。
    *   **用户名/昵称**：加粗显示。
    *   **简介 (Bio)**：简短的个人描述。
    *   **数据概览**：
        *   **关注数 (Following)**
        *   **粉丝数 (Followers)**
        *   **获赞数 (Likes Received)**
*   **操作按钮**：
    *   **当前用户**：显示“编辑资料”按钮。
    *   **其他用户**：显示“关注/取消关注”和“私信”按钮。

### 2.2 内容标签页 (Tabs)
页面中部提供 Tab 切换，包含以下模块：
1.  **作品 (Works/Posts)**：用户发布的所有照片/配方。
2.  **喜欢 (Likes)**：用户点赞过的帖子。
3.  **收藏 (Collections)**：用户收藏的帖子。
4.  **关于/器材 (About/Gear)**：详细的个人介绍和常用的相机/镜头清单。

## 3. 详细功能模块

### 3.1 资料编辑 (Edit Profile)
点击“编辑资料”弹出对话框或跳转页面：
*   **修改头像**：
    *   支持**本地上传**图片文件。
    *   **限制**：文件大小不超过 **5MB**。
    *   **格式**：支持 JPG, PNG, WEBP。
    *   **交互**：点击头像区域触发文件选择，上传后实时预览。
*   **修改昵称**：唯一性校验。
*   **修改简介**：支持多行文本。
*   **器材清单**：添加/删除常用的相机和镜头型号（用于自动补全）。

### 3.2 作品展示 (Works Gallery)
*   **布局**：复用首页的**瀑布流布局**。
*   **卡片样式**：
    *   展示缩略图。
    *   显示标题。
    *   **数据角标**：显示该作品的点赞数和浏览量。
    *   **状态标识**（仅自己可见）：如“审核中”、“私密”等状态。

### 3.3 喜欢与收藏 (Likes & Collections)
*   **布局**：同样采用瀑布流布局。
*   **隐私控制**：
    *   “喜欢”列表默认公开（可设置仅自己可见）。
    *   “收藏”列表默认仅自己可见（可设置公开）。

### 3.4 关注/粉丝列表 (Follows)
点击关注数或粉丝数时，弹出模态框展示列表：
*   **列表项**：头像 + 昵称 + 简介摘要 + 关注按钮（互相关注状态）。

## 4. UI/UX 规范 (基于 Element Plus)

### 4.1 颜色与风格
*   **主色调**：保持与全站一致（黑白灰为主，强调色为蓝色或品牌色）。
*   **留白**：保持页面清爽，突出图片内容。

### 4.2 交互细节
*   **Tab 切换**：使用 `el-tabs`，切换时无刷新加载内容（前端缓存或骨架屏 loading）。
*   **滚动加载**：作品列表支持无限滚动 (Infinite Scroll)。
*   **空状态**：当某个 Tab 下无内容时，使用 `el-empty` 展示友好的提示和插画。

## 5. API 需求

### 5.1 用户信息
*   `GET /api/v1/users/{user_id}`: 获取用户公开信息（含关注/粉丝数）。
*   `PUT /api/v1/users/me`: 更新个人资料。

### 5.2 内容列表
*   `GET /api/v1/posts?user_id={id}`: 获取某用户的作品列表。
*   `GET /api/v1/users/{id}/likes`: 获取某用户点赞的列表。
*   `GET /api/v1/users/{id}/collections`: 获取某用户收藏的列表。

### 5.3 社交关系
*   `POST /api/v1/interactions/follow/{user_id}`: 关注用户。
*   `DELETE /api/v1/interactions/follow/{user_id}`: 取消关注。
